# GitHub-User-Finder
Автор: Снеткова Александра Ивановна 

## Описание
Приложение для поиска пользователей GitHub через официальное API. Позволяет просматривать базовую информацию и сохранять выбранных пользователей в локальный список избранного.

## Функционал
- Поиск пользователей по логину.
- Отображение имени и ссылки на профиль.
- Сохранение списка избранного в JSON-файл.
- Валидация пустых запросов.

## Технологии
- Python 3.x
- Библиотека `requests` для работы с API.
- Библиотека `tkinter` для графического интерфейса.

## Использование API
Для поиска используется публичный эндпоинт: `https://github.com{username}`. Ключ API не требуется для небольшого количества запросов.

## Как запустить
1. Установите зависимости: `pip install requests`
2. Запустите скрипт: `python main.py`
