# 🔍 GitHub User Finder 🚀
Элегантное десктопное приложение на Python для мгновенного поиска информации о пользователях GitHub с возможностью сохранения в «Избранное».

---

## 🌟 Основные возможности

*   **⚡ Быстрый поиск:** Получение данных (имя, логин, ссылка на профиль) напрямую из GitHub API.
*   **📂 Избранное:** Сохранение интересных профилей в локальную базу данных (JSON).
*   **🎨 Удобный GUI:** Интуитивно понятный интерфейс, созданный на библиотеке `tkinter`.
*   **🛡️ Валидация:** Система защиты от пустых запросов и обработки сетевых ошибок.
*   **📊 Просмотр списка:** Весь список избранных пользователей отображается прямо в приложении.

---

## 🛠️ Стек технологий

- **Язык программирования:** Python 3.10+
- **Библиотека GUI:** `Tkinter` (встроена в Python)
- **Работа с сетью:** `Requests`
- **Хранение данных:** `JSON`

---

## 🚀 Как запустить проект

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com
   ```

2. **Установите необходимую библиотеку:**
   Для работы с API нам понадобится библиотека `requests`:
   ```bash
   pip install requests
   ```

3. **Запустите приложение:**
   ```bash
   python main.py
   ```
